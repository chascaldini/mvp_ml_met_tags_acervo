# Dataset

Este projeto utiliza dados públicos do acervo do Metropolitan Museum of Art (The Met).

## Fonte dos dados

Metropolitan Museum of Art Open Access Dataset  
https://github.com/metmuseum/openaccess

## Arquivo utilizado

- MetObjects.csv

## Forma de acesso

O dataset é carregado diretamente no notebook via URL (raw), garantindo reprodutibilidade e evitando dependência de arquivos locais.
